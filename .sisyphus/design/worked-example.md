# Architecture B worked example (n=4, t=3, seed=42)

This note fixes a tiny human-readable parameter set for the Architecture B reference model and records one deterministic transcript generated by `cargo run -p pvthfhe-bench --bin worked_example` with `seed=42`.

## Toy parameters

- `n_parties = 4`
- `threshold = 3`
- `N_ring = 8`
- `q = 97`
- `t_plain = 4`
- `Δ = floor(q / t_plain) = 24`
- secret distribution: ternary `{ -1, 0, 1 }`
- error and smudging distribution: small integers `{ -1, 0, 1 }`
- message: `m = [1, 0, 1, 0, 1, 0, 1, 0]`
- seed: `42`

All polynomial arithmetic is in the negacyclic ring `R_q = Z_q[X]/(X^8 + 1)`.

## Step 3 — Encrypt

The toy encryption follows the Architecture B formula:

```text
ct = (c0, c1)
c0 = pk · u + e1 + Δ·m (mod q)
c1 = a · u + e2 (mod q)
```

with deterministic samples:

```text
u  = [1, -1, 0, -1, -1, -1, 1, 1]
e1 = [1, 0, -1, 1, 0, 1, -1, 0]
e2 = [1, 1, -1, -1, 1, 0, 0, 1]
```

So the ciphertext is:

```text
c0 = [36, -29, 23, 32, -43, 9, 24, 41]
c1 = [3, -20, 42, 34, 4, -44, -3, 38]
```

## Step 5 — Aggregate

Aggregate the threshold subset:

```text
D = d_0 + d_1 + d_2 = [-9, 23, 1, -34, -30, -9, 2, -40] (mod 97)
```

Then decode via:

```text
m_recovered = Round((c0 + D) · t_plain / q) mod t_plain
            = [1, 0, 1, 0, 1, 0, 1, 0]
```

The binary asserts `m_recovered == m` and terminates only if the equality holds.
