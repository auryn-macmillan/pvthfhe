## 3. Critical/High Gap Details

### L1 — dkgRoot Reuse Deadlock (HIGH)

**Root cause**: `hermine.rs::generate_session` derives `session_id` (used as `dkgRoot` on-chain) as:

```rust
SHA256( participant_id_1 || participant_id_2 || ... || threshold )
```

This is deterministic. When the same committee retries DKG (e.g. after a network partition caused < t completions), they produce the **same dkgRoot**. Before this fix, `registerSession` would revert `AlreadyRegistered` with no recovery path.

**Fix**: Added `abortSession(bytes32 dkgRoot)` to `SessionRegistry.sol`:

- Sets `Session.aborted = true`
- Emits `SessionAborted` event
- `registerSession` now allows re-registration of aborted sessions (clears `aborted` flag)
- `markEpochConsumed` and `verifySession` reject aborted sessions with `SessionAbortedError`
- Epochs consumed pre-abort remain consumed (prevents cross-retry epoch replay)

**Security invariant preserved**: An aborted session cannot consume new epochs. Re-registration resets only the `aborted` flag; the `consumed` mapping is preserved, so epoch N consumed during round 1 cannot be replayed in round 2.

| `test_liveness_consumedEpochsNotReusableAfterReregister` | Consumed epochs persist across abort+re-register (no replay) |

## 5. Non-Issues Confirmed

- **fold.rs verifier**: No loop/memory exhaustion risk. All code paths bounded by `sequential_t` and `MAX_INSTANCE_BYTES`. Returns `Err` on malformed input.
- **adapter.rs batch_verify**: Returns `Err` on first failure; caller can retry. `MAX_BATCH_STMTS` caps iteration.
- **hermine.rs reconstruction**: Returns `Err` when shares < threshold; no stall.
- **No liveness gaps** found in `pvthfhe-cyclo` or `pvthfhe-nizk` crates.

---

## 6. Future Work (Deferred)

- **Session deadline / expiry**: Add a `deadline` block number field to `Session`; auto-expire sessions after deadline via an `expireSession` function. Requires governance/parameter decisions on epoch duration.
- **Committee supersede**: Explicit `supersede(oldDkgRoot, newDkgRoot)` to atomically rotate committees on-chain, preventing race conditions between old and new committees.
